Counter-Infos

Was muss ich tun, um einen Counter zu bekommen?

Als erstes müssen Sie sich anmelden. Das ist natürlich vollkommen kostenlos. Nach der Anmldung können Sie sich einen Counter aussuchen und auf Ihrer Internetseite einfügen. Die von Ihnen getroffenen Counter-Einstellungen werden auf unseren Servern gespeichert. Gehen jetzt Besucher auf Ihre Internetseite, so steigt die Besucherzahl sowohl auf den Servern, als auch auf der Ausgabe auf ihrer Internetseite. Die Ausgabe ist immer in Echtzeit, das heißt, Sie haben immer einen genauen Überblick über das aktuelle Besucherverhalten.

Wen zählt der Counter?

Wen genau der Counter zählt, hängt von Ihren Einstellungen ab, die Sie bei der Anmeldung gemacht haben. Haben Sie keine IP-Sperre eingestellt, so wird jeder Klick auf Ihre Seite gezählt, ganz egal, von wem er wann gemacht wurde. Haben Sie aber eine IP-Sperre gewählt, so hängt es von dem eingestellten Zeitintervall ab, ob ein Besucher einmal oder mehrmals gezählt wird. So kann man zum Beispiel einstellen, dass ein Benutzer mit seiner IP-Adresse nur einmal innerhalb eines Tages gezählt wird, egal wie oft er auf Ihre Seite geht.

Wie richte ich die IP-Sperre ein?

Bei Ihrer Anmeldung können Sie einstellen, ob sie eine IP-Sperre verwenden wollen oder nicht. Entscheiden Sie sich für eine IP-Sperre, können sie ein Zeitfenster angeben, in dem eine IP nur einmal gezählt werden soll. Besucht dann jemand Ihre Seite, so wird anhand der IP festgestellt, wann diese Person das letzte Mal die Seite besucht hat. Liegt der letzte Besuch innerhalb des von Ihnen bestimmten Zeitfensters, so wird er nicht gezählt. Liegt er außerhalb, so wird er gezählt und ein neues, gleichgroßes Zeitfenster tut sich auf, in dem seine weiteren Besuchen nicht gezählt werden.
